* Installation *

Non-Steam users:
- Extract into your ROTT95 directory.
- Type in SKOTOS at the command prompt.

Steam (3D Realms Anthology) users:
- Extract into your ROTT95 directory.
- Launch "Rise of the Triad: Dark War"
- Select #4 (Setup)
- Select "Use Modified Stuff"
- Select "Choose Alternate Game Levels"
- Select "SKOTOS.RTL"
- Go back to the main menu
- Select "Save and Run ROTT"
